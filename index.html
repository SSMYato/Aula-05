Propriedades Position

Existem cinco valores de posições diferentes

  • static
  • relative
  • fixed
  • absolute
  • sticky

Os elementos são posicionados usando propriedadas top, bottom, left e right.

Position "static"

Os elementos em HTML por padrão já são posicionados em static. Nessa posição eles não são afetados pelas propiedades, top, left, right e bottom.

Exemplo de "static"

Position "relative"

São afetados pelas propriedades top, left, right e bottom O conteúdo não se ajusta para se encaixar no espaço deixado pelo elemento

Exemplo de relative

Position "Fixed"

O elemento posicionado em relação a viewport, ou seja, ele vai permanecer no mesmo lugar da página.

Exemplo de "Fixed"

Position "absolute"

O Elemento é posicionadocom relação ao seu ancestral mais próximo

Exemplo de "absolute"

Position "sticky"

Um elemento sticky é posicionado com base na posição de rolagem do usuário. Ele alterna entre relative e fixed, depende da posição de rolagem

Exemplo de "sticky"
